## Sobre o Projeto

O **MetricNotes** é um sistema web open-source voltado para a autogestão de produtividade, construído sob a ótica de *Business Intelligence* (BI). 

Diferente das ferramentas tradicionais de *to-do list*, o MetricNotes ataca a raiz da desorganização através de uma dinâmica de despejo de ideias incentivando o usuário a registrar seu fluxo de pensamento bruto em blocos de notas e, a partir desse registro, categorizar as ideias em tarefas acionáveis.

O objetivo principal é mitigar a exaustão do trabalho remoto e promover a saúde mental, utilizando análise de dados para identificar períodos de maior desempenho e otimizar o fluxo de atividades, garantindo que o tempo livre seja uma consequência natural do esforço focado.

## Principais Funcionalidades

- **Brainstorming Inicial:** Blocos de anotações dedicados ao "despejo mental" de ideias e demandas.
- **Conversão de Tarefas:** Fluxo de UI/UX focado em transformar notas brutas em tarefas acionáveis (to-dos).
- **Dashboards Analíticos:** Painéis de BI pessoal com métricas de conclusão de tarefas.
- **Gráficos de Calor (Heatmaps):** Visualização intuitiva dos níveis de produtividade e prioridade, facilitando a tomada de decisão.
- **Análise de Desempenho:** Identificação automática dos horários e dias de maior rendimento do usuário.

## 🛠️ Tecnologias e Arquitetura

O projeto utiliza uma arquitetura Fullstack modular, separando a regra de negócios da interface de visualização.

**Back-end:**
- PHP
- Laravel (Framework)
- MySQL

**Front-end:**
- TypeScript
- React

**Metodologia e Gestão:**
- Design Thinking para estruturação de UI/UX.
- PMBOK (Project Management Institute) para gestão de tempo.
- Kanban (via Trello) para organização de tarefas e sprints.
